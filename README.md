# Converge-PG

Data merger for PostgreSQL

Built on top of [Converge](http://github.com/azanar/converge)


## Installation

Add this line to your application's Gemfile:

```ruby
gem 'converge-pg'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install converge-pg
